Orientações:

  • Utilize um smartphone para a realização desse teste.
  • Este teste simula o fluxo onde o usuário pagador estará acessando um E-commerce
  • As informações do QR Code gerado será o QR Code de exemplo apresentado no documento "Anexo I - Padrões para Iniciação do PIX" (Versão 1.2.006).
  • Para fins de testes os campos do QR Code "Nome do Recebedor" (Merchant Name) e "Valor" (Transaction Amount) poderão ser alterados nesta tela. Observação: a imagem do QR Code não se altera, ela está sendo exibida para fins de exemplificação.
  • A opção "Exibir código Pix" exibirá uma segunda alternativa de exibição onde os dados do QR Code poderá ser exibido. (As informações estão sendo apresentadas no formato Base64 para não expor de forma explicita as informações do QR Code)
  • Essa abordagem não afeta a experiência do usuário no que diz respeito a escolher o app de seu banco para poder receber o pix
  • O app para realização do teste pode ser encontrado no seguinte endereço: https://play.google.com/store/apps/details?id=br.com.testePI
